# App Icons

Place your app icons here:

- `icon.ico`  — Windows (256x256 recommended)
- `icon.icns` — macOS
- `icon.png`  — Linux (512x512 recommended)

You can generate all three from a single PNG using:
https://www.icoconvert.com  (for .ico)
https://cloudconvert.com    (for .icns)

Or use the `electron-icon-builder` npm package:
  npx electron-icon-builder --input=icon.png --output=build
